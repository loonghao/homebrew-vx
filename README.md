# Homebrew Tap for vx

[vx](https://github.com/loonghao/vx) is a Universal Development Tool Manager that helps you manage Node.js, Python, Go, Rust, and 30+ other development tools.

## Installation

```bash
brew tap loonghao/vx
brew install vx
```

Or install directly:

```bash
brew install loonghao/vx/vx
```

## Usage

```bash
# Check version
vx --version

# List available tools
vx list

# Install Node.js
vx install node@22

# Run npm directly (auto-installs if needed)
vx npm install
```
